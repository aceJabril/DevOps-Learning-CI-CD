# DevOps Learning – CI/CD

This repo contains my CI/CD assignments completed as part of my DevOps learning with CoderCo. Each task builds on the last, covering GitHub Actions, automated testing, Docker integration and CI/CD best practices.

## Assignments

| Task | Description | Link |
|------|-------------|------|
| Task 1 | Basic CI Pipeline – linting, unit tests, Docker build and push | [View Task 1](./task-1) |
| Task 2 | Full CI/CD Pipeline – Terraform EC2 provisioning, Flask app deployment, Docker Compose with Redis, automated deployment via SSH | [View Task 2](./task-2) |

## What This Repo Covers
- Setting up GitHub Actions workflows from scratch
- Automating tests and linting on every push
- Building and pushing Docker images through a pipeline
- Storing sensitive credentials securely using GitHub Secrets
- Provisioning cloud infrastructure with Terraform
- Automatically deploying applications to EC2 via SSH
- Running multiple containers with Docker Compose
- Debugging and fixing real pipeline errors
